# BMW Global Sales Analysis 2018-2025

## Project Overview
Analysis of BMW global sales data from 2018 to 2025 using SQL and Power BI.

## Tools Used
- SQL Server (SSMS) — Data Analysis & KPI Queries
- Power BI — Interactive Dashboard & Visualizations
- Excel — Data Validation

## Key Insights
- Total Revenue: €1,571bn
- Best Selling Model: iX
- Average Price Per Model: €63,977

## Dashboard Includes
- Sales Trend Over Years
- Revenue by Region
- Units Sold by Model
- BEV Share Growth 2018-2025
